# RISC-V Architecture Tests

This directory contains pre-compiled binaries (ELF files) used for integration testing of the `rsriscv_emu` emulator.

## Source

These binaries are compiled from the official [RISC-V Software Test Suite](https://github.com/riscv-software-src/riscv-tests).

* **Origin**: https://github.com/riscv-software-src/riscv-tests
* **License**: BSD-3-Clause (See original repository for details)

## Purpose

These artifacts are included in the repository to enable fast CI/CD execution without requiring a full RISC-V GNU Toolchain installation.
